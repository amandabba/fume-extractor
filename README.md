These are the files I adjusted based on Giovanni Aggiustatutto's fume extractor design (https://www.instructables.com/Soldering-Fumes-Extractor/), 
adjusted to accommodate a larger fan (140 × 140 × 38 mm) to have more power in preventing the inhalation of nasty fumes! 

This project was inspired from when I got a bad cough for weeks even since a bad soldering session in my room.
